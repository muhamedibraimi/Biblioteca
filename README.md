# Biblioteca
In una biblioteca, le pubblicazioni sono suddivise in libri e riviste. Ogni pubblicazione ha un titolo e un anno di pubblicazione, ma i libri hanno anche un autore e un numero di pagine, mentre le riviste hanno un numero progressivo e un mese di pubblicazione.
La gestione dei prestiti è effettuata tramite l’informazione che identifica se il libro è disponibile o meno.
Deve essere possibile ricercare una pubblicazione e conoscere le statistiche relative agli anni di pubblicazione (es. quante riviste sono pubblicate dal 2023…)
